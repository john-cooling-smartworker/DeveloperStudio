The DOCTYPE

The <!DOCTYPE html> declaration tells the browser that a given file is a markup .html file and that a set of web standards must be applied to it. It is the first line of code that goes at the top of every HTML or XHTML file.

Elements

There are six kinds of elements: void; template; raw text; escapable raw test; foreign, and normal

LINKS

Linking in CSS files

CSS files are often kept in a "ccs" or "style" director as part of the site. This is preferred to including CSS styling in the html file because it is easier see, and control when it's a seperate file

< link href="pathname of the CSS file" type="text/css" rel="stylesheet" />

CSS links are defined in the < head > section of the html page. It is common to define multiple CSS files, one after another in the < head > section. When styling in the files clash eg <h1> styling in all files, the styling is applied in the order the files are defined. The last < h1 > tag found in the last file will override earlier tags

Creating navigation links. The < a > element

Links are also a core feature of Internet navigation. They power movement from one page to another, thats surfing the web!

The <a> HTML element (or anchor element), with its href attribute, creates a hyperlink to web pages, files, email addresses, locations in the same page, or anything else a URL can address. To link to different website you use an absulute URL. Linking to other pages on the same website you don't have to specifiy the domain name you can use shorthand known as relative URL.

Relative URL

If all the pages of the site are in the same folder then the value for the href attribute is just the file name.

To link to a file in same folder, with text descriptor of 'Adding CSS styling to elements'

<a href="Styling.html">Adding CSS styling to elements<a>

To link to a file in a child folder, with text descriptor of 'Getting started in DEVOPS'

<a href="devops/introduction.html">Getting started in DEVOPS<a>

To link to a file in a grandchild folder, with text descriptor of 'Using CPanel'

<a href="devops/cpanel/cpanel.html">Using CPanel<a>

To link to a file in a parent folder, with text descriptor of 'Australian Shares Home'

<a href="../index.html">Australian Shares Home<a>

To link to a file in a grandparent folder, with text descriptor of 'Smartworker'

<a href="../../index.html">Smartworker<a>

EMAIL links

Create a link that starts up the users email program and specifies a to: address

<a href="mailto:brigalow@smartworker.com">Email Brigalow<a>

Open links in a new window

To link to a another site in a parent folder, with text descriptor of 'Australian Shares' and open it in a new browser window. Use the target keyword

<a href="../index.html">Australian Shares Home<a target="_blank">Australian Shares<a>

Linking to a specific part of the same page

First you must add an id="target name" to the elements where you want to land. The id's target name must be unique on the page and can only start with an underscore or letter

<h2 id="navlinks">Creating navigation links<h2>

To create links to to your landing element you use the <a> element and the href attribute starts with a # symbol.

<a href="#navlinks">Creating navigation links<h2>

Linkeing to a specific part of another page or website.

Use exactly the same technique as for same page links. the two differences are

  • the target must have an id
  • you must use the correct absolute URL or relative url

    Table Tags

    Tag Name Description
    <table> Table The wrapper element for all HTML tables.
    <thead> Table Head The set of rows defining the column headers in a table.
    <tbody> Table Body The set of rows containing actual table data.
    <tr> Table Row The table row container.
    <td> Table Data The table row container.
    <tfoot> Table Foot The set of rows defining the footer in a table.

    Table Attributes

    Attribute Name Description
    colspan Column Span Defines how many columns a td element should span.
    rowspan Row Span Defines how many rows a td element should span.
