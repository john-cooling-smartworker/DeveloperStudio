Profile

John Cooling

In the Beginning

Started programming in 1974. Yes, that's fifty years programming experince!

First coded banking applications using IBM Assempler. Writing the programs out long hand on coding sheets. Coding sheets turned into punch cards by enormous data preparation teams, over fifty people. Cards feed into a IBM 370 computer to assemble, link, and execute.

Errors were corrected by mentally assembling the code fix and manually hand puching it into a "link deck" for rerun. With this method you saved so much time, days, getting a rerun without having to reassemble the whole program.
