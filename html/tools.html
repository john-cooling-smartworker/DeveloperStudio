Country Wisdom

So many tools for programmers, which to pick?

These are the tools I use. Once I learn a tool I stick with until I hit the frustration boundary. Only then do I upgrade to something more modern. Why donm't I keep up with the latest tools often? Simple answer is, switching costs!

Tools

Basic Tool Fundamentals

  • The first thing is a code editor. There are many, Visual Studio, Atom, Vim, are just three. Select Visual Studio as your first editor, it has many addons available for all modern languages.
  • Second up you need to talk directly with your coding computer on a one to one basis. That requires a terminal to be able to interact with the command line. Its a Command Line Interface (CLI). Terminal is built into the MacOS. You can go the extra mile and get your self iTerm
  • There will be code and lots of it! To keep your code under control and packaged into workable bundles you need a source control management system. Everyone uses Git as the engine to drive you SCM and GitHub to store youe code in the cloud for easy access and sharing with others
